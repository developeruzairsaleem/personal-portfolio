# Uzair Saleem — Portfolio

> The fastest way to ship your SaaS without hiring a full team.

Personal portfolio for Uzair Saleem, full-stack SaaS developer. Targets founders who need a fast, reliable developer.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Testing:** Jest + React Testing Library
- **Deploy:** Vercel

## Local Dev
```bash
npm install
npm run dev
# Open http://localhost:3000
```

## Run Tests
```bash
npm test
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Push to GitHub
2. Import repo at vercel.com
3. Deploy (zero config needed)

Or use Vercel CLI:
```bash
npx vercel --prod
```
